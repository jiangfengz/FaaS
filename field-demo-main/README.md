# API任务执行插件

这是一个飞书多维表格字段捷径插件，用于执行各种API任务。

## 功能特性

- **API凭证绑定**: 安全的API Key输入和管理
- **任务选择**: 支持多种任务类型（情感分析、文本摘要、翻译、实体提取等）
- **信息输入**: 灵活的任务参数输入
- **实时执行**: 调用后端API并返回执行结果
- **错误处理**: 完善的错误提示和状态反馈

## 安装和部署

### 1. 安装依赖

```bash
npm install
```

### 2. 本地调试

```bash
npm run start
```

### 3. 打包发布

```bash
npm run pack
```

将生成的 `output/output.zip` 文件上传到飞书多维表格字段捷径平台。

## 使用说明

### 字段配置

插件包含三个核心配置字段：

1. **API凭证绑定**
   - 类型：密码输入框
   - 功能：输入API Key进行身份验证
   - 验证：基本的API Key格式验证

2. **任务选择**
   - 类型：下拉选择框
   - 选项：情感分析、文本摘要、翻译、实体提取
   - 功能：选择要执行的任务类型

3. **信息输入**
   - 类型：多行文本输入框
   - 功能：输入任务相关的文本信息
   - 限制：最大5000字符

### 执行流程

1. 用户在字段中配置API Key、选择任务类型、输入相关信息
2. 插件将配置信息发送到后端API
3. API处理任务并返回结果
4. 插件将执行结果显示在多维表格中

### 返回结果

插件返回Object类型字段，包含以下属性：

- **任务ID**: 唯一标识符（隐藏字段）
- **执行状态**: 成功/失败状态
- **执行结果**: 任务处理结果
- **执行时间**: 任务耗时（毫秒）
- **错误信息**: 错误详情（仅在失败时显示）

## API集成

### 请求格式

插件会向配置的API端点发送POST请求：

```json
{
  "api_key": "用户输入的API Key",
  "task": "选中的任务类型",
  "input": "用户输入的文本信息",
  "timestamp": 时间戳
}
```

### 响应格式

API应返回以下格式的JSON响应：

```json
{
  "task_id": "任务唯一标识",
  "status": "执行状态",
  "result": "处理结果",
  "execution_time": 执行耗时
}
```

### 错误处理

插件支持以下错误状态：

- **401**: API凭证无效
- **402**: 余额不足
- **5xx**: 服务器错误
- **其他**: 任务执行失败

## 开发说明

### 项目结构

```
src/
  └── index.ts          # 主入口文件
test/
  └── index.ts          # 测试文件
package.json            # 项目配置
tsconfig.json           # TypeScript配置
config.json             # 本地调试配置
```

### 自定义扩展

要添加新的任务类型，需要：

1. 在 `formItems` 的 `task_selection` 选项中添加新任务
2. 在 `i18n` 配置中添加对应的多语言文本
3. 确保后端API支持新的任务类型

### 安全注意事项

- API Key以密码形式输入，确保安全性
- 建议在生产环境中使用HTTPS协议
- 定期更新API Key以提高安全性
